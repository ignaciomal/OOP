Programación orientada a objetos

La programación orientada a objetos(OOP) diseña y construye software a partir de objetos que son instanciados en una clase, permitiendo definir atributos y comportamientos(metodos y funciones); por lo tanto su principal carcteristica es que soporta objetos. La OOP trabaja ciertos temas para poder usar dichos objetos, estos son: herencia, polimorfismo, abstracción y encapsulamiento.

Herencia

Se trata de un mecanismo quepermite heredar Las caracteristicas de una clase llamada clkase padre o tras clases llamadas clases hijas

Polimorfismo

son las diferentes acciones que puede realizar un objeto a un mismo mensaje o a la llamada a un metodo

Abstracción

Nos permite ocultar la complejidad interna de un objeto y exponer solo imformación esencial para su uso

Encapsulamiento

Restringe el acceso directo a los datos de un objeto y controla el accesomodificando la información a traves de metodos
